Наше веб-приложение "SiteChecker" предназначено для проверки и
выявления мошеннических и нежелатенльных веб сайтов с целью
предостережения пользоваьтелей интернета.
Функционал приложения:
    - Сравнение доменов сайтов с популярными
    - Анализ содержимого веб-страницы
    - Проверка отзывов о сайте у независимого источника
    - Составление пользовательского рейтинга доменов
    - Создание аккаунтов пользователей
    - Оценка пользователем доменных имен

В качестве платформы для реализации проекта был выбран "Anvil",
поскольку данный веб-ресурс для создания сайтов имеет простой конструктор форм,
использует в качестве backend'а один из самых популярных и простых в освоении
языков (Python), а также весьма простой в освоении API
